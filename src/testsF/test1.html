Cuestionario de ortografía

  1. 1. Selecciona el signo de puntuación adecuado para separar los siguientes elementos de una enumeración:

    "manzanas_ peras_ plátanos_ y uvas."





  2. 2. ¿Cuál es el signo de puntuación correcto para introducir una cita textual en el siguiente enunciado?

    "Como dijo Albert Einstein, 'La imaginación es más importante que el conocimiento'."





  3. 3. Escoge el signo de puntuación adecuado para indicar una pausa mayor que la coma pero menor que el punto y seguido en el siguiente enunciado:

    "El viaje fue largo, cansado_ sin embargo, valió la pena."





  4. 4. ¿Cuál es el signo de puntuación correcto para enfatizar lo que viene después de él en la siguiente oración?

    "No hay duda alguna_ el trabajo duro da resultados."





  5. 5. Selecciona el signo de puntuación adecuado para separar los enunciados que forman un párrafo en el siguiente texto:

    "El sol brillaba intensamente_ El cielo estaba despejado_ Las aves cantaban alegremente_"





Resultado

Respuestas correctas:

  1. 1. La respuesta correcta es el inciso b, la coma se utiliza para separar los elementos de una enumeración, como en este caso, donde se enumeran diferentes tipos de frutas.
  2. 2. El inciso b es la opción correcta: Los dos puntos se utilizan para introducir una cita textual, como en este caso, donde se cita una frase atribuida a Albert Einstein.
  3. 3. La opción correcta corresponde a la opción d Recuerda que el punto y coma se utiliza en este caso para indicar una pausa mayor que la coma pero menor que el punto y seguido, separando dos partes de un enunciado relacionadas entre sí.
  4. 4. Los dos puntos se utilizan para enfatizar lo que viene después de ellos, como en este caso, donde se enfatiza la afirmación que sigue a los dos puntos, por lo que b) es la respuesta correcta.
  5. 5. El punto y aparte se utiliza para separar los enunciados que forman un párrafo, indicando el final de uno y el inicio de otro., por lo que la b es la opción correcta.
